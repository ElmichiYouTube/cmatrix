# CMatrix
Un matrix para Windows escrito en C#

CMatrix es un **programa para Windows** de **Matrix** creado en **C#**. Si quieres compilarlo tú mismo, tienes que tener Windows 7 con la actualización más reciente hasta el Windows 11. O verificar si tienes el Framework de .NET 4.0.30319 en tu PC. Aunque el script ya verifica eso.
## Descarga
Si quieres descargar CMatrix, ve al <a href="https://github.com/ElmichiYouTube/cmatrix/releases/tag/C%23">Release</a> más reciente.
## Compilar
**Pasos para compilarlo**
1. Descarga `CMatrix.cs` y `Compile.cmd` en tu PC.
2. Ejecuta `Compile.cmd` y el script, solito va a compilarlo. Debería aparecer una carpeta llamada **bin** en el directorio donde ejecutó el script compilador, donde estará su **CMatrix.exe** listo para ejecutarse.
### Compilar manualmente
**En caso de que el compilador tenga un error, puede hacerlo manualmente.**

Si su PC es de 32 bits, escriba `SET DIR=C:\Windows\Microsoft.NET\Framework\v4.0.30319\csc.exe` en la consola.

Si su PC es de 64 bits, escriba `SET DIR=C:\Windows\Microsoft.NET\Framework64\v4.0.30319\csc.exe` en la consola. O también puede usar la de `C:\Windows\Microsoft.NET\Framework\v4.0.30319\csc.exe` ya que viene incluída en Windows de 64 bits.

**Si ya tiene todo listo, escriba lo siguiente...**
1. Escriba en la consola `%DIR% "CMatrix.cs"` o reemplaze `CMatrix.cs` por el nombre correcto del archivo C# por si lo cambió.
2. En unos segundos (usualmente 5, dependiendo de su hardware) va a aparecer su **CMatrix.exe** listo para ejecutarse.
